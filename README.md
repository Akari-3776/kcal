# kcal
# 🍎 カロリー計算アプリ

食べたカロリーと、1日の目標カロリーとの差を簡単に計算できるシンプルな Web アプリです。  
スマホでも PC でも動作し、GitHub Pages にそのまま公開できます。

---

## 🚀 機能

- 食べ物の **1個あたりのカロリー** を入力
- **食べた個数** を入力
- **1日の目標カロリー** を入力
- 「計算する」ボタンで  
  → **合計カロリー**  
  → **目標との差（不足 or 超過）**  
  を自動表示

## 📱 使い方

1. 「1個あたりのカロリー」を入力  
2. 「食べた個数」を入力  
3. 「目標カロリー」を入力  
4. 「計算する」を押す  
5. 結果が画面に表示されます
